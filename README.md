## 📌 Sobre o projeto

O **Life OS** é um sistema completo de organização pessoal desenvolvido como projeto de TCC e peça de portfólio. A aplicação foi projetada para unificar o gerenciamento da vida do usuário, integrando **Agenda Inteligente**, **Sistema de Metas** e **Controle Financeiro** em uma única plataforma focada em produtividade e evolução constante.

---

## 🧠 Funcionalidades Principais

### 📅 Agenda Dinâmica
* **Rotina Automatizada:** Adaptação automática baseada no dia da semana.
* **Gestão de Tarefas:** Checklists interativos com status de conclusão.
* **Eventos Livres:** Inserção de compromissos personalizados fora do padrão diário.

### 🎯 Sistema de Metas Conectadas
* **Prazos Estruturados:** Divisão visual entre metas de Curto, Médio e Longo prazo.
* **Gamificação:** O progresso aumenta automaticamente conforme as tarefas diárias são concluídas.
* **Cálculo Financeiro:** Metas monetárias calculam automaticamente a porcentagem atingida com base no valor guardado vs. valor alvo.

### 💰 Controle Financeiro (Database Local)
* **Fluxo de Caixa:** Registro detalhado de receitas (+) e despesas (-).
* **Categorização:** Filtro por Alimentação, Lazer, Estudo, Investimentos, entre outros.
* **Extrato e Saldo:** Atualização em tempo real do balanço geral.

### 📊 Dashboard Interativo
* **Visão 360º:** Resumo diário de todas as métricas do sistema.
* **Gráficos Profissionais:** Representação visual de gastos e ganhos.
* **Indicador de Produtividade:** Barra de progresso diário baseada no cumprimento da agenda.

## 🛠️ Tecnologias Utilizadas

* **Front-end:** HTML5, CSS3, JavaScript (Vanilla)
* **Design/UI:** Glassmorphism, CSS Variables, Design Responsivo (Mobile First)
* **Armazenamento:** `localStorage` (Web Storage API)
