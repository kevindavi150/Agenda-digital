# 🚀 Life OS - Sistema de Organização Pessoal

## 📌 Sobre o projeto
O **Life OS** é um sistema completo de organização pessoal desenvolvido como projeto de TCC.

Ele integra:
- 📅 Agenda inteligente
- 🎯 Sistema de metas
- 💰 Controle financeiro

Tudo em uma única plataforma, com foco em produtividade e evolução pessoal.

---

## 🧠 Funcionalidades

### 📅 Agenda
- Rotina automática por dia da semana
- Tarefas com status (concluído ou não)
- Eventos personalizados

### 🎯 Metas
- Curto, médio e longo prazo
- Progresso automático
- Metas financeiras com cálculo

### 💰 Finanças
- Registro de receitas e despesas
- Categorias (alimentação, lazer, investimento...)
- Saldo atualizado automaticamente

### 📊 Dashboard
- Visão geral do sistema
- Gráfico de gastos
- Progresso do dia

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Chart.js
